## Abstract

Matt from Pinata walks through Pinata Agents and how the platform supports agent-oriented workflows. The session covers practical uses for hosted agent infrastructure, file and data handling, and how developers can connect Pinata's tooling into AI-enabled applications.

## Transcript Summary

Matt demos the current Pinata Agents app, showing how users can create a custom agent, choose presets, configure personality and direction, attach skills, manage workspace files, and provide secrets or environment variables before the agent boots. The walkthrough focuses on making hosted agent setup less brittle and easier to standardize.

The session also covers provider support, including Anthropic, OpenAI, and OpenRouter, plus plans to support more of what OpenClaw-style workflows expect. Feedback from RaidGuild is treated as part of the product loop, with several recent changes coming directly from user testing.
